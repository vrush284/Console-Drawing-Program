## Credit Suisse - Take Home Exercise
-Language used is Javascript


## Installation
- npm install
This command will install all the dependencies


## Run the Project
- npm start
This command will Run the project

## Run all the test cases
- npm run test
This command will Run all the test cases

## Usage
-Command to create a canvas: C w h

-Commands to draw on canvas:
    - Draw line: L x1 y1 x2 y2
    - Draw Rectangle: R x1 y1 x2 y2
    - Fill Canvas: B x y colour

-Command to quit project: Q
 ** Note - All commands are case sensitive **
